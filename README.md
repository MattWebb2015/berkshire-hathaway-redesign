# Berkshire Hathaway Website Redesign

Final group project for Intro to Web Development.

This project redesigns the Berkshire Hathaway website with improved navigation, readability, visual hierarchy, organization, and responsive layout.

## Pages
- Home
- Reports
- News
- Companies
- Governance

## Project Scope

This project is a static redesign prototype for an Intro to Web Development final project. It focuses on improving the layout, navigation, readability, organization, and responsive design of the original Berkshire Hathaway website using representative content.
